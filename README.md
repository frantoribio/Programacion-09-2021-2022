# Programación - 09 Gestión de Bases de Datos mediante código

Tema 09 Gestión de Bases de Datos mediante código. 1DAM. Curso 2021/2022.

![imagen](https://www.mindinventory.com/blog/wp-content/uploads/2019/10/Offshore-app-developer.png)

## Contenidos
1. Bases de datos Relacionales
2. El Desfase Objeto-Relacional
3. Acceso a Base de Datos Relacionales
4. CRUD con JDBC
5. El problema de Null y el tipo Optional
6. Procesando información en colecciones. API Stream y programación funcional
